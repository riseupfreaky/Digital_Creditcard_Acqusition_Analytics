
# Digital Credit Card Acquisition Analytics

## 📌 Project Overview
This project analyzes omnichannel acquisition for a digital credit card across 
Paid Search, Paid Social, Affiliate, Retail Partners, and App Store traffic.

## 🎯 Business Questions
- Which channels drive the most efficient approvals?
- Where are the biggest funnel drop-offs?
- What is the ROI of each channel?
- How much uplift do retail partners generate?
- How should budget be allocated across channels?

## 🧩 Project Components
1. Funnel Analysis  
2. Channel Performance  
3. Attribution Models  
4. A/B Testing  
5. Budget Optimization  
6. Tableau Dashboard  

## 📁 Repository Structure
- `/data/raw` — raw datasets  
- `/data/processed` — cleaned datasets  
- `/notebooks` — analysis notebooks  
- `/src` — Python scripts  
- `/dashboards` — Tableau files  
- `/docs` — project plan, notes  

## 📊 Deliverables
- Funnel report  
- Attribution comparison  
- Experiment results  
- Optimization model  
- Executive dashboard
